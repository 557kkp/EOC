# WELearnToSleeep

WeLearn刷学习时长   
本软件遵守GPLv3协议，且免费、开源，禁止售卖！ 

[Github](https://github.com/Avenshy/WELearnToSleeep)


## 简介

* 输入账号密码一键学习
* 可以自行选择课程
* 多线程学习，同时学习整个单元
* 可以指定时间或设置随机时间


**PS:本软件只增加学习时长，如需完成课程请食用[WELearnToSleep(WeLearn一键完成课程)](../一键完成/简介.md)**
