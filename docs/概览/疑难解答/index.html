反馈问题#

如果希望解决问题，建议通过Github提出Issue，如果能够提供账号就更好了(需要复现BUG，或者实现新功能)

不建议在greasy fork反馈，很少会去看

info

值得注意的是，WELearnHelper中的留言功能是单向的，我并没有办法与你进一步交流
