3 款网课助手的安装大体相同，Unipus 网课助手额外支持 Chrome 扩展

-   安装最新版本的 Chrome
-   油猴
    -   安装最新版本的 Tamper Monkey
    -   安装最新版本的完整版脚本(支持单元测试)
        -   可以在 Github 右侧 Release 下载 UnipusHelper[版本号].user.js 文件
        -   油猴理应自动弹出
        -   如果油猴未弹出，在油猴中新建脚本，并复制粘贴保存
    -   安装最新版本的轻量版脚本(无需绑定)(不支持单元测试)
        -   通过[GreasyFork](https://greasyfork.org/zh-CN/scripts/405123)安装
-   Chrome 扩展
-   在 Github 上找不到下载的地方？
    -   点击页面右侧的 Release，.user.js 结尾即为油猴脚本，.crx.zip 结尾，即为 chrome 扩展
